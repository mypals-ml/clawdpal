### Local File Server
1.  **Prepare Folder**: Create a folder for test images.
    ```bash
    mkdir -p ~/clawdpal
    ```
2.  **Add Data**: Save a test image named `test.jpg` inside that folder.
3.  **Start Server**:
    ```bash
    cd ~/clawdpal
    python3 -m http.server 8000
    ```
4.  **Verify**: Open `http://localhost:8000/test.jpg` in your browser.
